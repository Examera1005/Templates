Modal Components

Comprehensive modal system with multiple types and configurations

Basic Modals

Simple modals with different sizes and configurations.

const modal = new Modal({ size: 'medium', type: 'default' }); modal.setContent(` <div class="modal-header"> <h3 class="modal-title">Basic Modal</h3> </div> <div class="modal-body"> <p>This is a basic modal example.</p> </div> `); modal.open();

Alert Modals

Pre-built alert modals for different message types.

Modal.alert('Operation completed successfully!', { title: 'Success', variant: 'success', onOk: () => console.log('Alert closed') });

Confirmation Modals

Interactive confirmation dialogs with promise-based responses.

Modal.confirm('Are you sure you want to delete this item?', { title: 'Delete Item', danger: true, confirmText: 'Delete', cancelText: 'Cancel' }).then(confirmed => { if (confirmed) { console.log('Item deleted'); } });

Form Modals

Dynamic form modals with validation and data collection.

Modal.form({ fields: [ { type: 'text', name: 'name', label: 'Name', required: true }, { type: 'email', name: 'email', label: 'Email', required: true }, { type: 'textarea', name: 'message', label: 'Message' } ] }, { title: 'Contact Us', submitText: 'Send Message' }).then(data => { console.log('Form data:', data); });

Custom Content Modals

Modals with custom HTML content and interactive elements.

const modal = new Modal({ size: 'large' }); modal.setContent(document.getElementById('custom-content').innerHTML); modal.open();

Data Attribute Usage

Trigger modals using data attributes without JavaScript.

<button data-modal-trigger="open" data-modal-target="#template-modal" data-modal-size="medium"> Open Modal </button> <div id="template-modal" class="modal-content-template"> <div class="modal-header"> <h3 class="modal-title">Template Modal</h3> </div> <div class="modal-body"> <p>This modal was opened using data attributes!</p> </div> </div>

Data Attribute Modal

This modal was opened using data attributes! No JavaScript required.

The modal system automatically handles the opening and closing based on the data attributes.
