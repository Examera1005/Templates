How to Use

🚀 Easy Integration

Simply include the CSS files and start using the classes!

<link rel="stylesheet" href="animations.css"> <link rel="stylesheet" href="svg-animations.css"> <!-- Then use the classes --> <button class="btn-animated btn-gradient">Beautiful Button</button> <div class="card-animated animate-float">Floating Card</div>

📋 Files Included

  • animations.css - Complete animation library with buttons, cards, and effects
  • svg-animations.css - SVG-specific animations and graphics
  • index.html - This demo page with all examples
